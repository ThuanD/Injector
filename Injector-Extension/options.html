📖 Hướng dẫn sử dụng
Dành cho người mới — không cần biết code
⚠️
Cảnh báo bảo mật — Đọc trước khi dùng
Script JavaScript có thể đọc mọi nội dung trên trang, bao gồm mật khẩu, cookie, thông tin thẻ ngân hàng. Chỉ chạy script từ nguồn bạn tin tưởng.
  • Không paste code từ người lạ gửi qua mạng xã hội, Zalo, Telegram
  • Không chạy script trên trang ngân hàng, ví điện tử, trang đăng nhập
  • Nên dùng ChatGPT để giải thích code trước khi chạy nếu không chắc
Injector làm được gì?
Tự động chạy JavaScript trên bất kỳ trang web nào mỗi khi bạn mở trang đó. Dùng để ẩn quảng cáo, tự điền form, theo dõi giá, nhận thông báo Telegram… và rất nhiều thứ khác — không cần biết lập trình.
🚀 Cách 1 — Dùng Template có sẵn (dễ nhất)
1
Vào trang Templates
Click 📋 Templates ở sidebar trái. Bạn sẽ thấy danh sách các script có sẵn theo từng nhóm chức năng.
2
Xem trước và chọn template
Nhấn nút 👁 (preview) để xem code, hoặc nhấn ＋ Use Template để dùng ngay. Không cần hiểu code.
3
Điền URL Pattern rồi lưu
Chỉ cần điền một thứ: URL Pattern — trang nào bạn muốn script này chạy. Ví dụ: *.tiki.vn hay *.youtube.com. Nhấn ＋ Add Script là xong.
🎯 URL Pattern là gì?
💡
URL Pattern quyết định trang nào script sẽ chạy. Dùng dấu * như một ký tự "đại diện cho bất kỳ thứ gì". Không cần điền chính xác từng URL — chỉ cần đúng domain là đủ.
Pattern bạn điền Ý nghĩa Ví dụ URL khớp
*.youtube.com Mọi trang trên YouTube youtube.com/watch?v=...
www.youtube.com/shorts/...
*.tiki.vn/product/* Chỉ trang sản phẩm Tiki tiki.vn/product/abc-p123.html
https://docs.google.com/* Mọi Google Docs docs.google.com/document/d/...
*.facebook.com Toàn bộ Facebook www.facebook.com, m.facebook.com
* Chạy trên mọi trang web Bất kỳ URL nào
🤖 Cách 2 — Nhờ ChatGPT / Gemini viết script
1
Copy một trong các prompt mẫu bên dưới
Chọn prompt phù hợp với việc bạn muốn làm, nhấn Copy Prompt. Thay phần trong ngoặc [...] bằng mô tả của bạn.
2
Paste vào ChatGPT hoặc Gemini và gửi
Nhấn ↗ Mở ChatGPT để mở tab mới. Paste prompt vào và gửi. AI sẽ trả về đoạn code hoàn chỉnh.
3
Copy code → Tạo script mới trong Injector
Copy phần code AI trả về, quay lại đây nhấn ＋ New Script, dán vào ô JavaScript Code, điền URL Pattern rồi lưu.
💡 Mẹo hay
🔄
Script không chạy? Thử reload trang sau khi lưu script. Một số trang cần load lại để extension nhận diện được.
🧪
Test nhanh không cần reload: Mở popup extension → tìm script → nhấn ▶ Run để chạy thử ngay lập tức mà không cần reload trang.
📦
Backup scripts: Vào ⚙ Settings → ↓ Export JSON để lưu tất cả scripts ra file. Dùng ↑ Import để khôi phục hoặc chuyển sang máy khác.
🔔
Nhận thông báo Telegram: Các script Monitoring cần BOT_TOKEN và CHAT_ID của bot Telegram. Hỏi ChatGPT: "How do I create a Telegram bot and get my chat ID?"
