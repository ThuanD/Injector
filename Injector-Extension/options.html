📖 User Guide
For beginners — no coding required
⚠️
Security Warning — Read Before Use
JavaScript scripts can read all content on the page, including passwords, cookies, banking information. Only run scripts from sources you trust.
  • Do NOT paste code from strangers via social media, Zalo, Telegram
  • Do NOT run scripts on banking, e-wallet, or login pages
  • Should use ChatGPT to explain code before running if unsure
What can Injector do?
Automatically run JavaScript on any website when you open that page. Use it to hide ads, auto-fill forms, track prices, get Telegram notifications… and much more — no programming required.
🚀 Method 1 — Use Ready Templates (Easiest)
1
Go to Templates Page
Click 📋 Templates in the left sidebar. You'll see a list of ready-to-use scripts grouped by function.
2
Preview and Select Template
Press 👁 (preview) to view code, or press ＋ Use Template to use immediately. No need to understand the code.
3
Fill URL Pattern and Save
Only need to fill one thing: URL Pattern — which pages you want this script to run on. Example: *.tiki.vn or *.youtube.com. Press ＋ Add Script and you're done.
🎯 What is URL Pattern?
💡
URL Pattern determines which pages the script will run on. Use * as a wildcard for anything. No need to enter exact URLs — just the domain is enough.
Pattern you enter Meaning Example matching URLs
*.youtube.com All pages on YouTube youtube.com/watch?v=...
www.youtube.com/shorts/...
*.tiki.vn/product/* Only Tiki product pages tiki.vn/product/abc-p123.html
https://docs.google.com/* All Google Docs docs.google.com/document/d/...
*.facebook.com Entire Facebook www.facebook.com, m.facebook.com
* Run on all websites Any URL
🤖 Method 2 — Ask ChatGPT / Gemini to Write Script
1
Copy One of the Sample Prompts Below
Choose a prompt that matches what you want to do, press Copy Prompt. Replace the part in brackets [...] with your description.
2
Paste into ChatGPT or Gemini and Send
Press ↗ Open ChatGPT to open a new tab. Paste the prompt and send. AI will return complete code.
3
Copy Code → Create New Script in Injector
Copy the code AI returns, come back here and press ＋ New Script, paste into the JavaScript Code field, fill URL Pattern and save.
💡 Pro Tips
🔄
Script not running? Try reloading the page after saving the script. Some pages need to reload for the extension to recognize them.
🧪
Quick test without reload: Open extension popup → find script → press ▶ Run to test immediately without reloading the page.
📦
Backup scripts: Go to ⚙ Settings → ↓ Export JSON to save all scripts to a file. Use ↑ Import to restore or transfer to another device.
🔔
Telegram notifications: Monitoring scripts need BOT_TOKEN and CHAT_ID from your Telegram bot. Ask ChatGPT: "How do I create a Telegram bot and get my chat ID?"
