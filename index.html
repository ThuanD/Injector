FAQ
Frequently Asked Questions.
💸 Is this extension really free?
Completely free, open source. No paid features, no ads, no data collection.
🔒 Is this extension safe?
Open source on GitHub, you can read every line of code before installing. Scripts only run on sites you specify — don't auto-run everywhere.
🏪 Why not publish on Chrome Web Store?
Google charges developers $5 to publish on store. This extension is completely free so we share directly via GitHub. Manual install only takes 2 minutes.
🔄 Does it auto-update?
File-based extensions don't auto-update. When new version is released, download ZIP from GitHub and repeat step 4 (Load unpacked).
🤖 Can I use it without coding knowledge?
Yes. Extension has 20+ ready-to-use templates. If you need custom scripts, there are built-in prompts to help ChatGPT write them — just describe in Vietnamese.
⚠️ What if scripts don't run correctly?
Open DevTools (F12) → Console tab to see errors. Every script has [Injector] logs for debugging. Or use the ▶ Run button in popup to test without page reload.
📱 Does it work on Firefox?
Currently only supports Chrome and Chromium-based browsers (Edge, Brave, Opera…). Firefox uses different APIs so not yet supported.
💾 Where are scripts stored?
Stored in chrome.storage.local — only on your device. Use Export JSON feature in Settings to backup and transfer to other devices.
