Cài đặt
4 bước,
dưới 2 phút.
Không cần Chrome Web Store. Tải thẳng từ GitHub, cài tay là xong.
1
Tải file ZIP từ GitHub
Vào trang GitHub của extension, nhấn nút xanh Code → Download ZIP. Hoặc click thẳng vào link bên dưới.
↓ Download ZIP
2
Giải nén file ZIP
Chuột phải vào file vừa tải → Extract All (Windows) hoặc double-click (macOS). Bạn sẽ có một thư mục Injector-main.
3
Bật Developer Mode trong Chrome
Mở Chrome, vào chrome://extensions. Bật công tắc Developer mode ở góc trên bên phải màn hình.
⚙ Mở chrome://extensions
4
Load thư mục extension
Nhấn Load unpacked → mở vào thư mục Injector-main vừa giải nén → chọn tiếp thư mục con Injector-Extension. Extension sẽ xuất hiện ngay trong danh sách và icon ⚡ hiện trên toolbar.

FAQ
Câu hỏi thường gặp.
💸 Extension này có miễn phí không?
Hoàn toàn miễn phí, mã nguồn mở. Không có tính năng trả phí, không có quảng cáo, không thu thập dữ liệu của bạn.
🔒 Extension có an toàn không?
Mã nguồn công khai trên GitHub, bạn có thể đọc từng dòng code trước khi cài. Script chỉ chạy trên các trang bạn chỉ định — không tự động chạy trên tất cả mọi nơi.
🏪 Sao không đăng lên Chrome Web Store?
Google thu phí developer $5 để đăng lên store. Extension này hoàn toàn miễn phí nên chúng tôi chia sẻ thẳng qua GitHub. Cài tay chỉ mất 2 phút.
🔄 Có tự cập nhật không?
Extension cài từ file không tự cập nhật. Khi có phiên bản mới, bạn tải lại ZIP từ GitHub và làm lại bước 4 (Load unpacked) là xong.
🤖 Không biết code thì có dùng được không?
Được. Extension có 20+ template dùng ngay. Nếu cần script riêng, có sẵn prompt mẫu để nhờ ChatGPT viết — bạn chỉ cần mô tả bằng tiếng Việt là được.
⚠️ Script chạy không đúng thì sao?
Mở DevTools (F12) → tab Console để xem lỗi. Mỗi script đều có log [Injector] để debug. Hoặc dùng nút ▶ Run trong popup để chạy thử mà không cần reload trang.
📱 Có dùng được trên Firefox không?
Hiện tại chỉ hỗ trợ Chrome và các browser Chromium-based (Edge, Brave, Opera…). Firefox dùng API khác nên chưa hỗ trợ.
💾 Script lưu ở đâu?
Lưu trong chrome.storage.local — chỉ trên máy bạn. Dùng chức năng Export JSON trong Settings để backup và chuyển sang máy khác.
